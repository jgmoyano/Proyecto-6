PROYECTO N°6 - JUAN MOYANO ANFOSSI

El proyecto consiste en un portafolio personal que resume el trabajo realizado durante todo el Bootcamp en una pagina web simple, atractiva visualmente y de facil navegación.
La pagina cuenta con una descripcion profesional, el link a los 5 proyectos anteriores, una zona de contacto y la posibilidad de descargar mi CV.
Para la realización de este proyecto se utilizo HTML, CSS y JAVASCRIPT.
